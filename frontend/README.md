# Airbnb Clone (Django + React)

## Backend

- Django REST Framework
- JWT Authentication
- Apps: users, properties, bookings, reviews, payments

## Frontend

- React (Vite)
- Axios
- React Router
- JWT Auth

## Run Backend

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
